# api-sandbox

A simple Node.js REST API sandbox for testing and development.

## Endpoints

- `GET /items` — list all items
- `GET /items/:id` — get item by id
- `POST /items` — create item
- `PATCH /items/:id` — update item
- `DELETE /items/:id` — delete item

## Getting Started

```bash
npm install
node index.js
```
